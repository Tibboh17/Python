# Idea
- Use bubble sort.

# Code
```python
numlist = []

for _ in range(5):
    numlist.append(int(input()))

for i in range(4):
    for j in range(i+1, 5):
        if numlist[i] > numlist[j]:
            numlist[i], numlist[j] = numlist[j], numlist[i]
        
avg = int(sum(numlist)/5)
mid = numlist[2]

print(avg)
print(mid)
```
