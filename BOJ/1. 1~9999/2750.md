# Idea
- Use bubble sort.

# Code
```python
n = int(input())
numlist = []

for _ in range(n):
    numlist.append(int(input()))

for i in range(n-1):
    for j in range(i+1, n):
        if numlist[i] > numlist[j]:
            numlist[i], numlist[j] = numlist[j], numlist[i]
        
for num in numlist:
    print(num)
```
